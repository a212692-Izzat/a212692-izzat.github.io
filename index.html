Jetpack Compose

Google's modern native UI toolkit for Android, used to build declarative UIs using Kotlin.

Room Database (AndroidX)

An abstraction layer over SQLite that simplifies database access with full type-safety in Android.

Retrofit 2 + OkHttp

HTTP client library for Android for REST API integration with robust error handling.

Firebase (Firestore + Authentication)

Google's backend platform for real-time cloud data storage and secure, scalable user authentication.

Modern Android camera library and Google's on-device machine learning suite for image and text analysis.
